# 🚇 교통 혼잡도를 고려한 대중교통 루트 안내 AI 서비스 (CAT)

> **팀명:** CAT (Create AI Tailored to you)  
> **프로젝트 기간:** 2026.04 ~ 2026.06  

기존 대중교통 길 찾기 서비스의 한계를 보완하여, **실시간 혼잡도와 시간대별 교통 상황을 반영한 최적의 우회 및 대안 루트를 제공하는 AI 기반 네비게이션 서비스**입니다. 단순히 최단 거리를 찾는 것을 넘어 출퇴근 시간대 만석 광역버스나 혼잡한 지하철 구간을 피한 '실제 탑승 가능한 현실적인 경로'를 추천합니다.

---

## 👥 팀원 소개 및 역할
- **임민정 (전체 총괄 & 프론트엔드):** UI/UX 피그마 설계, 프론트엔드 개발 및 UI 구현
- **김효린 (프론트엔드):** UI/UX 피그마 제작, 화면 컴포넌트 개발 및 연동
- **구민진 (백엔드):** 공공데이터 수집, 데이터 전처리, 데이터베이스 설계
- **전소이 (백엔드):** Flask 서버 구축, RESTful API 설계 및 Swagger 연동
- **정세연 (백엔드):** 버스/지하철 승하차 데이터 수집 및 데이터 분석

---

## 🧠 주요 AI 기술 및 알고리즘
1. **Random Forest (혼잡도 분류 모델)**
   - 정류장별/시간대별 누적 승객 데이터를 바탕으로 현재 노선의 혼잡 상태를 분류합니다.
2. **LSTM (시계열 승객 수 예측 모델)**
   - 과거 시간대의 승하차 인원 흐름 추이를 학습하여, 특정 미래 시간대의 예상 탑승객 수를 시계열로 추정합니다.
3. **혼잡도 가중치 기반 최적 경로 알고리즘**
   - ODsay API 노선망 데이터를 가공하여, AI 모델이 예측한 **혼잡도 패널티 가중치**(`weight = 기본시간 + 혼잡패널티`)를 노선 그래프에 반영해 최적의 대안 루트를 계산합니다.
